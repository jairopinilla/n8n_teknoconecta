# Checklist pre-publicacion

Usar antes de publicar cualquier contenido (anuncio, mensaje masivo, red social, blog).

## Contenido

- [ ] Datos verificados contra Stays
- [ ] No sobrepromete
- [ ] No usa palabras prohibidas (lujo, unico, imperdible, etc.)
- [ ] Beneficios son concretos y reales
- [ ] Restricciones mencionadas claramente
- [ ] Tono: sobrio, confiable, claro

## Seguridad

- [ ] No contiene Wi-Fi, codigos ni claves
- [ ] No contiene tokens ni credenciales
- [ ] No contiene datos personales de huespedes
- [ ] No contiene informacion del edificio que deba ser interna

## Fotos

- [ ] Son reales
- [ ] Sin distorsion extrema
- [ ] Muestran lo que el huesped encontrara
- [ ] No muestran espacios cuyo uso no esta confirmado

## Multi-idioma

- [ ] Version ES lista
- [ ] Version EN lista (localizada, no traducida literal)
- [ ] Version PT-BR lista (localizada para Brasil)
