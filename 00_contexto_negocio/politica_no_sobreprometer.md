# Politica: No sobreprometer

## Regla de oro

Todo contenido publicado (anuncio, mensaje, red social, web) debe reflejar exactamente lo que el huesped encontrara al llegar.

## Checklist de verificacion

- [ ] El amenity existe fisicamente en la unidad
- [ ] La distancia esta medida, no estimada
- [ ] El espacio comun esta confirmado por reglamento o Stays
- [ ] No se usaron palabras como "lujo", "unico", "imperdible", "espectacular", "increible"
- [ ] No se prometio estacionamiento
- [ ] No se publicaron claves, codigos ni datos sensibles
- [ ] No se mostraron fotos de espacios que no se entregaran
- [ ] La tarifa refleja la realidad (impuestos, cargos adicionales)

## Espacios comunes del edificio (confirmados)

> Residentes, sujeto a disponibilidad y coordinacion con administracion.
- Sala cowork
- Gimnasio al aire libre (rooftop)
- Quinchos / sala gourmet
- Lavanderia de autoservicio

## NO disponibles

- Piscina — no hay
- Estacionamiento propio — no hay (publico pago a una cuadra: Saba Paseo Bulnes)

## Tono permitido

Claro, concreto, sobrio, confiable. Beneficios reales sin adornos.
