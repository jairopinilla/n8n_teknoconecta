# Glosario

| Termino | Significado |
|---------|-------------|
| ADR | Average Daily Rate — tarifa promedio por noche |
| RevPAR | Revenue Per Available Room — ingreso por noche disponible |
| OTA | Online Travel Agency — plataforma de reservas (Airbnb, Booking, etc.) |
| PMS | Property Management System — sistema de gestion de propiedad |
| CRM | Customer Relationship Management |
| RAG | Retrieval Augmented Generation — patron de IA con busqueda en base de conocimiento |
| MCP | Model Context Protocol — protocolo de herramientas para agentes de IA |
| n8n | Orquestador de workflows (self-hosted) |
| Stays.net | PMS principal del negocio |
| PriceLabs | Herramienta de pricing dinamico |
| Directus | Backend operacional (CMS/headless) |
| Neon | PostgreSQL serverless con pgvector |
| Supabase | Plataforma de base de datos (actualmente en desuso) |
